<h1>TaskGuru - Project Management Software</h1>

TaskGuru is a robust and intuitive project management software designed to streamline your team's workflow and enhance collaboration. With a focus on simplicity and efficiency, TaskGuru provides powerful tools for managing tasks, tracking progress, and facilitating communication within your projects.

<h2>Features</h2>

Task Management: Create, assign, and prioritize tasks with ease. Keep track of your team's progress and deadlines effortlessly.

User-friendly Interface: Intuitive design for a seamless user experience, making it easy for team members to navigate and use TaskGuru efficiently.

Kanban Board: Visualize your project's progress using a Kanban board. Move tasks across different stages to represent their status, from "To Do" to "Done."

Collaborative Environment: Foster collaboration among team members with real-time updates and comments on tasks. Stay informed and engaged throughout the project lifecycle.

Customizable Workflows: Tailor TaskGuru to match your team's unique workflow. Define custom task statuses, priorities, and other parameters to align with your project requirements.

Calendar Integration: Plan and schedule tasks with an integrated calendar feature. Keep everyone on the same page by visualizing project timelines and milestones.

<h2>Technologies Used</h2>
React: Frontend development for a responsive and dynamic user interface.

Node.js: Backend development to handle server-side logic and API integrations.

MongoDB: Database management for storing and retrieving project data efficiently.
